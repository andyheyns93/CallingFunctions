# CPP - DLL HOOK

1. Run the console app 

* `CallingFunctionsConsole.exe`

2. Run the injector

* `CallingFunctionsInjector.exe`
* This will inject 'CallingFunctionsHook.dll' into 'CallingFunctionsConsole.exe'
